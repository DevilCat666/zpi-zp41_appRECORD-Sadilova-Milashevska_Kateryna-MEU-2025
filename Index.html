Опис предметного середовища

Онлайн-магазин ювелірних прикрас - це веб-платформа, призначена для продажу ювелірних виробів через інтернет. Система надає можливість користувачам переглядати каталог товарів, здійснювати покупки, відстежувати статус замовлень та отримувати персоналізовані рекомендації.

Основні функції системи:

  • Каталог прикрас з детальними описами та фотографіями
  • Система пошуку та фільтрації товарів
  • Особистий кабінет користувача
  • Кошик покупок та оформлення замовлень
  • Система відгуків та оцінок
  • Інтеграція з платіжними системами

Тема, мета та розташування

Тема:

Створення веб-сайту онлайн-магазину ювелірних прикрас

Мета:

Створити сучасний та зручний онлайн-магазин прикрас, який дозволить клієнтам переглядати асортимент виробів, замовляти товари, слідкувати за трендами у світі прикрас

Посилання на репозиторії:

    Посилання на репозиторій з виконання Web.
    Посилання на репозиторій зі Звітування.

Структура документа

Веб-застосунок складається з наступних основних HTML-файлів:

  • index.html - головна сторінка з каталогом товарів
  • product.html - сторінка окремого товару
  • cart.html - сторінка кошика
  • account.html - особистий кабінет

HTML-код таблиці

Приклад таблиці категорій товарів:


<table class="categories-table">
    <thead>
        <tr>
            <th>Категорія</th>
            <th>Кількість товарів</th>
            <th>Ціновий діапазон</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Каблучки</td>
            <td>50+</td>
            <td>1000-5000 грн</td>
        </tr>
        <tr>
            <td>Сережки</td>
            <td>100+</td>
            <td>800-3000 грн</td>
        </tr>
    </tbody>
</table>

Теорія про теги таблиць:

  • <table> - контейнер для таблиці
  • <thead> - шапка таблиці
  • <tbody> - тіло таблиці
  • <tr> - рядок таблиці
  • <th> - заголовок стовпця
  • <td> - клітинка даних

HTML-код форми

Форма оформлення замовлення:


<form class="order-form" action="/submit-order" method="POST">
    <div class="form-group">
        <label for="name">Ім'я:</label>
        <input type="text" id="name" name="name" required>
    </div>
    
    <div class="form-group">
        <label for="email">Email:</label>
        <input type="email" id="email" name="email" required>
    </div>
    
    <div class="form-group">
        <label for="phone">Телефон:</label>
        <input type="tel" id="phone" name="phone" required>
    </div>
    
    <button type="submit">Оформити замовлення</button>
</form>

HTML-код зображення

Приклад додавання зображення товару:


<img 
    src="images/ring-gold.jpg" 
    alt="Золота каблучка з діамантом" 
    class="product-image"
    width="400" 
    height="400"
    loading="lazy"
>

Атрибути тегу img:

  • src - шлях до файлу зображення
  • alt - альтернативний текст
  • width - ширина зображення
  • height - висота зображення
  • loading - метод завантаження
  • class - CSS клас для стилізації
